# database設計

## Usersテーブル
|Column|Type|Options|
|------|----|-------|
|nickname|string|null:false|
|email|string|null:false, unique:true|
|password|string|null:false|

- has_one :credit_card
- has_one :profile
- has_many :items
- has_many :orders
- has_many :comments
- has_many :likes
- has_many :evaluations
